# procon29-kyogi
第29回全国高等専門学校競技部門 沖縄高専のリポジトリ

## すべてのモジュールをビルド

```sh
./build.sh
```

## すべてのモジュールを起動

```sh
./start.sh
```

## 動作環境

Ubuntu 18.04
